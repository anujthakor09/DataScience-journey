# Problem:
Single Number

# Platform:
LeetCode #136

# Difficulty:
Easy

# Approach:
Used BITWISE operator XOR to solve as 0^0 = 0, 0^1 = 1...
this cancels out same numbers and only the single one remains

# Time Complexity:
O(n)

# Space Complexity:
O(1)

# First_thought:
using maps storing the key value and checking linearly
