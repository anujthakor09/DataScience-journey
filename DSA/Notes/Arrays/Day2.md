# Important Array Notes
**Important Points(Vector)**
  * Dynamic in nature : 
  * Uses STL as tool box
  * Vector are the STL container
  * We need to include vector library to use vectors
  * Vectors allocated with the memomry at the run time
  * Allocated in Heap

**Basic Syntax**
  * vector <int> vec;
  * vector <int> vec = {1,2,3};
  * vector <int> vec = (3,0)

**Vector Functions**
  * size = vec.size()
  * vec.push_back()
  * vec.pop_back()
